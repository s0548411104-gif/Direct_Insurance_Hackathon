# 🛡️ DirectVision - אימות נכס דיגיטלי מבוסס AI

DirectVision היא מערכת חיתום חכמה מקצה לקצה (End-to-End Underwriting Engine) לחברות ביטוח. המערכת מונעת הונאות, חוסכת זמן וכסף, ומייתרת את הצורך בסוקרי שטח פיזיים על ידי שילוב של מודלי ראייה ממוחשבת מתקדמים ולוגיקה עסקית.

## 🚀 הבעיה והפתרון

**הבעיה:** תהליך אישור פוליסות ביטוח כיום דורש הוצאת סוקר לשטח. התהליך גורר עלויות כוח אדם אדירות, מעכב את אישור הפוליסה בשבועות, ופותח פתח להונאות ודיווחים שקריים העולים לחברות הביטוח מיליארדים בשנה.

**הפתרון שלנו:**
הלקוח מצלם את הנכס דרך הסמארטפון מתי שנוח לו. המערכת:
1. מאמתת את המיקום (GPS) ואיכות התמונה בזמן אמת.
2. מכווצת את התמונה ושומרת אותה בענן.
3. מריצה סוכני AI (Vision & Logic) המנתחים את הנכס, משווים את הממצאים להצהרת הלקוח (CRM), ומקבלים החלטת חיתום - **הכל בתוך 120 שניות**.

## ✨ פיצ'רים מרכזיים (Highlights)

* 🕵️ **מנגנון מניעת הונאות (Fraud Shield):** המערכת מזהה ופוסלת אוטומטית העלאת תמונות שאינן קשורות לנכס (תמונות מהאינטרנט, חיות מחמד, או פלייסחולדרים של חסימות רשת).
* ⚖️ **מגשר ההצהרות (The Reconciler):** אלגוריתם חכם המשווה בין ממצאי הראייה הממוחשבת (למשל: זיהוי עסק בבית) לבין הפוליסה של הלקוח ב-CRM. המערכת "סולחת" ללקוחות שהצהירו אמת ומונעת קנסות שווא.
* ⚡ **אופטימיזציית ביצועים (Edge Computing):** דחיסת תמונות חכמה בצד הלקוח החוסכת עד 70% ממשקל הקובץ, מאיצה את זמני הטעינה וחוסכת עלויות ענן ורוחב פס.
* 📊 **דאשבורד חתם בזמן אמת:** מסך ניהול המציג ציון חיתום מחושב (0-100), רדאר סיכונים, ו"יומן ראיות" (Audit Trail) מפורט המסביר את קבלת ההחלטות של ה-AI.

## 🛠️ טכנולוגיות וארכיטקטורה (Tech Stack)

המערכת בנויה בארכיטקטורת **Microservices**:

* **Frontend (קצה הלקוח והדאשבורד):** React.js, Vite, TypeScript.
* **Backend (שרת התורים):** Node.js, Express, Socket.io (לעדכוני זמן אמת), Multer.
* **Database & Storage:** MongoDB Atlas, Cloudinary.
* **AI Engine (המוח בחיתום - FastAPI):** * Python, Uvicorn.
  * **Google Gemini 1.5 Flash:** משמש כ"עיניים" (Vision Agent) לניתוח הויזואלי של החדרים.
  * **Anthropic Claude 3.5 Sonnet:** משמש כ"חתם הראשי" (Underwriter Agent) לשקלול נתונים, הערכת סיכונים וגישור מול ה-CRM.

    CLAUDE_API_KEY=your_anthropic_key_here

    node server.js
